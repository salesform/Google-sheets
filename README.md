# Használatuk

Nyisd meg a Google Sheets-t, majd kattints a Bővítmények > Apps Script menüpontra.
Illeszd be a kiválasztott kódot
Ments és futtasd. Add meg a szükséges engedélyeket és ha kell időzítsd
